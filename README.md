# Sunset Drift — Unity WebGL test build

This repo hosts the Unity engine port of Sunset Drift (Phase 0: drivable Golden Mile).
Play: https://nomansky1.github.io/sunset-drift-unity/

Controls — desktop: WASD/arrows drive, SPACE drift, SHIFT boost.
Phone (landscape): touch = auto-throttle, left/right half = steer,
bottom-left corner = brake/drift, bottom-right corner = boost.

Source project: `Downloads/SunsetDriftUnity` (local). The original Three.js game stays live at
https://nomansky1.github.io/sunset-drift-v2/
